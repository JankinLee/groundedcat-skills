# 世界观契合度检查清单

在创作新章节、引入新设定或修改已有规则时，使用此清单确保内容与世界规则一致。

## 1. 规则系统一致性

### 力量/技术/魔法系统
- [ ] **新能力符合体系分类**：若存在念能力六系、修仙境界、魔法学派等分类，新能力是否归属明确？
- [ ] **代价平衡**：强大能力是否伴随相应代价或限制？
- [ ] **规则边界**：是否触及或突破了既定的规则边界？
- [ ] **学习曲线合理**：角色获得能力的速度与方式是否符合世界观的学习机制？

### 社会与文化背景
- [ ] **社会阶层反映**：角色的言行是否符合其社会地位和文化背景？
- [ ] **禁忌与习俗**：新情节是否违反已设定的禁忌或习俗？
- [ ] **语言与称谓**：对话中的称谓、敬语、方言是否符合世界设定？
- [ ] **经济与政治**：角色行为是否考虑世界观中的经济体系和政治结构？

## 2. 时空与地理一致性

### 时间线
- [ ] **时间流逝合理**：章节内时间流逝与前后章节是否连贯？
- [ ] **历史事件参照**：提及的历史事件是否与已设定时间线一致？
- [ ] **年龄与成长**：角色年龄变化是否与时间流逝匹配？

### 地理与空间
- [ ] **地点存在性**：新出现的地点是否在世界地图或地点列表中？
- [ ] **距离与移动时间**：角色移动所需时间是否与距离、交通方式匹配？
- [ ] **环境特征**：地点描述是否与已设定的气候、地形、植被一致？

## 3. 人物与关系一致性

### 角色行为
- [ ] **动机符合性格**：角色的新行为是否与其既定性格、背景、欲望一致？
- [ ] **能力匹配**：角色展现的能力是否与其之前的能力水平相符？
- [ ] **关系发展自然**：角色关系变化是否有足够的铺垫和触发事件？

### 群体动态
- [ ] **势力关系**：群体间的互动是否符合已设定的势力关系网？
- [ ] **组织行为**：门派、公会、国家等组织的行为是否符合其宗旨和资源？
- [ ] **冲突合理性**：群体冲突的根源是否在世界观中有合理依据？

## 4. 物品与科技一致性

### 物品属性
- [ ] **物品功能**：新物品的功能是否与世界观中的科技/魔法水平匹配？
- [ ] **稀有度**：稀有物品的出现频率是否合理？
- [ ] **制造逻辑**：物品的制造方式是否在现有技术/能力范围内？

### 科技水平
- [ ] **技术迭代**：新技术是否在现有科技基础上合理发展？
- [ ] **知识传播**：知识的获取和传播方式是否符合世界观？
- [ ] **资源限制**：科技应用是否受限于世界观中的资源条件？

## 5. 多世界融合一致性（针对融合题材）

### 世界交互规则
- [ ] **融合接口**：不同世界间的连接方式是否符合已设定的融合机制？
- [ ] **规则冲突处理**：当不同世界规则冲突时，是否有明确的处理原则？
- [ ] **能量/物质转换**：跨世界能量、物质转换是否遵循守恒或代价原则？

### 风格协调
- [ ] **美学融合**：不同世界的视觉、听觉、文化元素是否协调？
- [ ] **叙事语调**：多世界叙事的语调是否保持一致或有意切换？
- [ ] **读者认知管理**：多世界信息是否以读者可理解的方式呈现？

## 6. 检查流程

### 写前检查（规划阶段）
1. 列出本章计划引入的新设定元素
2. 对照世界观文档，逐一检查一致性
3. 记录潜在冲突，规划调整方案

### 写中检查（创作阶段）
1. 每完成一个场景，检查该场景涉及的世界观元素
2. 特别注意对话中隐含的文化、社会信息
3. 确保环境描写与地点设定一致

### 写后检查（修订阶段）
1. 通读全章，标记所有世界观相关元素
2. 使用此清单逐项检查
3. 如有冲突，决定：调整内容、调整世界观记录、添加例外说明

## 7. 冲突处理原则

### 优先级
1. **核心规则不可违反**：力量体系基础、时空结构、关键历史事件
2. **次要规则可有限调整**：社会细节、物品属性、地理特征
3. **新增例外需记录**：任何对规则的例外必须在世界观文档中明确记录

### 记录要求
- 所有世界观调整必须在对应的世界观文档中记录（`02-项目管理.md` 或 `03-世界观与时间线.md`）
- 例外规则需注明原因、适用范围、可能影响
- 重大调整需用户确认

## 8. 快速检查表（每章必查）

- [ ] 本章是否引入了新能力/技术？若引入，是否符合体系分类？
- [ ] 本章是否出现了新地点？若出现，是否与地理设定一致？
- [ ] 本章角色行为是否与其既定性格/背景相符？
- [ ] 本章是否涉及时间跳跃？若涉及，时间计算是否正确？
- [ ] 本章是否提及历史事件？若提及，细节是否准确？
- [ ] 本章对话中的文化元素（称谓、敬语、习俗）是否恰当？
- [ ] 本章物品/装备的功能是否与世界观科技/魔法水平匹配？
- [ ] 如为多世界融合，本章的世界交互是否符合融合规则？
